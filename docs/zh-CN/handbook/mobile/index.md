# 移动端

<PluginInfo name="mobile" deprecated=true></PluginInfo>

## 介绍

提供移动端页面配置，内核框架基于 Ant Design Mobile 实现，提供了各种扩展点，兼容添加部分桌面端区块。

## 安装

预置插件，需要先激活才能使用。

![20240712113500](https://static-docs.nocobase.com/20240712113500.png)
