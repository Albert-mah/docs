# 数据加载方式

## 介绍

设置区块数据加载方式有两种默认方式：一种是自动加载，在页面加载时自动展示数据，适用于无需用户干预即可展示数据的情况；另一种是筛选后加载，只有在用户进行筛选并有条件时才触发数据加载，适用于需要用户交互才展示数据的场景，能够提升页面性能和用户体验

### 筛选后才加载数据

适用于交互式筛选和动态加载数据的场景（区块联动）。它默认不加载数据，只有在用户进行了筛选并且有了筛选条件时才会触发数据加载。
