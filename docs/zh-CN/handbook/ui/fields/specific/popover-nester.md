# 子表单（弹窗）

对于关系层级复杂、字段众多关系表情况下，子表单布局可能显得冗长且杂乱，难以有效呈现数据的主次关系。

子表单（弹窗）则针对这种场景，将关系表单以弹窗形式显示，同时支持在子表格中使用

- 标题字段：当弹窗关闭时显示的字段信息
