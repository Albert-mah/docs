# 本地化管理

<PluginInfo name="localization-management"></PluginInfo>

## 介绍

本地化管理插件用于帮助管理和实现 NocoBase 的本地化过程。对 NocoBase 的菜单、数据表、字段等等以及所有的插件进行翻译，以适应某一特定地区的语言和文化。

## 安装

内置插件，无需额外安装。
